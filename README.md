# SMS Spam Dataset Preprocessing (Programming for AI)

## Overview
This project focuses on preprocessing a text-based SMS dataset to prepare it for AI and machine learning applications such as spam detection.

## Dataset
- Source: UCI Machine Learning Repository
- Type: Text (SMS messages)
- Total messages: 5,574
- Labels:
  - ham (normal message)
  - spam (unwanted or scam message)

## Preprocessing Steps
- Removal of duplicate messages
- Conversion of text to lowercase
- Removal of URLs, punctuation, and numbers
- Tokenization of text
- Removal of stopwords
- Lemmatization of words
- Saving the cleaned dataset

## Output
- raw_sms_spam.csv
- cleaned_sms_spam.csv

## Applications
- SMS spam filtering
- Fraud detection systems
- Email and chat moderation
